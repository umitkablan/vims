
SOURCE A VIM SCRIPT RANGE
=========================
http://vim.wikia.com/wiki/Source_current_file_when_editing_a_script

Yank the vimscript you want to execute and fire:
    :@"
This is much more simple than.. write a function.. in that function get a temporary file.. write the range to file.. and execute delete..
    function! SourceRange() range
      let tmpsofile = tempname()
      call writefile(getline(a:firstline, a:lastline), l:tmpsofile)
      execute "source " . l:tmpsofile
      call delete(l:tmpsofile)
    endfunction
    command! -range Source <line1>,<line2>call SourceRange()
